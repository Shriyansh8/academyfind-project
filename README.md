# AcademyFind

A responsive full-stack coaching institute listing platform built with React, Node.js, Express, MongoDB, and Leaflet Maps.

## Features
- Institute listings
- Search & category filters
- Real-time distance calculation
- Interactive maps
- User authentication
- Responsive UI
- Payment modal integration

## Tech Stack
Frontend: React + Vite + TailwindCSS  
Backend: Node.js + Express  
Database: MongoDB Atlas  
Maps: Leaflet + OpenStreetMap

## Run Frontend
cd frontend
npm install
npm run dev

## Run Backend
cd backend
npm install
npm run server
